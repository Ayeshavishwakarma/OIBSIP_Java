# Online Reservation System (Java)

## Objective
To develop a console-based application for booking and managing train tickets.

## Features
- User login authentication
- Ticket reservation with PNR number
- Ticket cancellation using PNR
- View all reservations

## Technologies Used
- Java
- Scanner class
- ArrayList
- Iterator

## How it Works
1. User logs in with ID and password
2. Selects options from menu
3. Can reserve, cancel, or view tickets

## Outcome
This project helped in understanding Java concepts like OOP, loops, and data handling.
