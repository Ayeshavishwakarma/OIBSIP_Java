# OIBSIP Java Internship Projects

This repository contains Java-based console applications developed as part of the OIB-SIP Internship.

---

## 1. Number Guessing Game

### Objective
To create a simple game where the user guesses a randomly generated number.

### Features
- Generates random number between 1 to 100
- User enters guesses
- Provides hints (too high / too low)
- Limited attempts
- Displays win or loss message

### Technologies Used
- Java
- Random class
- Scanner class

### Outcome
This project helped improve logical thinking and understanding of loops and conditional statements.

---

## 2. Online Reservation System

### Objective
To develop a system for booking and managing train tickets.

### Features
- User login authentication
- Ticket reservation with PNR number
- Ticket cancellation using PNR
- View all reservations

### Technologies Used
- Java
- Scanner class
- ArrayList
- Iterator

### Outcome
This project helped in understanding real-world application flow and data handling in Java.
